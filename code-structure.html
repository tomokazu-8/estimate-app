検出された問題点
⚠ excel-loader.js にコードが重複
externalDbLoaded, showDbOverlay, closeDbOverlay, updateDbStatus, loadExcelDB がファイル内で2回定義されています（150行目以降が完全な重複）。
⚠ CAT_RATIOS が data.js と calc-engine.js で二重定義
data.js (行9) と calc-engine.js (行72) で別々の値が定義されており、後から読み込まれる calc-engine.js の値で上書きされます。
⚠ exportEstimate() が未実装
Excel出力ボタンは存在しますが、関数はトースト表示のみです。
⚠ グローバルスコープに全変数・関数が展開
ES Modules未使用のため、全ての変数と関数がwindowスコープに存在し、名前衝突のリスクがあります。
